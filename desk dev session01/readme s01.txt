### README Первая сессия разработки DESK Агента

#### Введение
Данный документ описывает ход первой сессии разработки десктопного приложения DESK Агент. Основная цель сессии заключалась в создании базового, работоспособного каркаса приложения, который в будущем будет служить средой исполнения для автономного ИИ-агента. Работа велась в соответствии с принципами Lean-методологии, фокусируясь на минимально необходимой функциональности для каждого шага.

#### Итог сессии
К концу первой сессии была создана основа для десктопного приложения, соответствующая заявленным архитектурным принципам. В частности
 Разработан каркас приложения на Python с использованием `PyQt6`.
 Реализована интерактивная рабочая область с адаптивной сеткой, функциями панорамирования и масштабирования.
 Создан базовый класс для будущих интерактивных блоков.
 Реализована логика взаимодействия с AI-агентом в отдельном потоке, используя `RAG` для выбора инструментов.

Следующий шаг — создание первого конкретного типа фрейма `TextFrame`, который позволит агенту отображать и редактировать текстовый контент.
